# GLauncher

Лаунчер для Garry's Mod с управлением серверами, встроенными модами и интеграцией со Steam.

## Возможности

- Запуск игры и подключение к серверам через Steam.
- Отображение статуса серверов (игроки, карта, режим, IP).
- Управление категориями и серверами (добавление, редактирование, удаление).
- Встроенные моды с установкой в папку `garrysmod/addons`.
- Discord Rich Presence: статус с названием сервера и IP:портом.
- Сворачивание в системный трей при запуске игры.
- Информация о Steam‑аккаунте (SteamID64, Steam2, аватар, имя).
- Импорт и экспорт конфигурации серверов.

## Как добавить сервер

1. Откройте лаунчер.
2. Слева выберите категорию (или создайте новую кнопкой **+**).
3. В центральной панели нажмите **+** (Добавить сервер).
4. Заполните поля:
   - **Название** — любое имя для сервера.
   - **Ссылка GameMonitoring** — вставьте URL с `gamemonitoring.ru` (например, `https://gamemonitoring.ru/garrys-mod/servers/11327153`). Лаунчер сам получит IP, порт и статус.
   - **Описание** (необязательно).
   - **Изображение** (необязательно) — выберите картинку для правой панели.
5. Нажмите **Сохранить**. Сервер появится в списке.

Теперь сервер можно запускать, перетаскивать и он будет отображаться в Discord.

## Установка и запуск (разработка)

1. Установите [Node.js](https://nodejs.org/) (рекомендуется версия 18 или выше).
2. Запустите Setup файл и установите лаунчер
3. Запустите лаунчер и настройте все для удобства
